# A-PIANO (Arduino Piano)

Aim : This is a mini which involves buzzers for producing different sounds.

## Materials required:
- Arduino UNO
- SparkFun Pushbutton switch 12mm
- Passive Buzzer
- Resistor 221 ohm
- Jumper wires (generic)
- UTSOURCE Electronic Parts
